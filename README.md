# Bank Management System

A simple Bank Management System built using Python and JSON for data storage.

## Features

- Create Account
- Deposit Money
- Withdraw Money
- Check Balance
- View Account Details
- Update Account Information
- Delete Account
- Account Authentication using Account Number and PIN
- JSON-based Database

## Technologies Used

- Python
- JSON
- Object-Oriented Programming (OOP)

## Project Structure

```
bank_management_project.py
data.json
README.md
```

## How to Run

1. Clone the repository
2. Run the Python file

```bash
python bank_management_project.py
```
